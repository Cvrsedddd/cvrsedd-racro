# Hypixel Racro

## Installation

```bash
run npm i
run npm install pm2 -g
```

## How to

Fill out .env
```
WEBHOOK_URL=
SSID=
```
Open CMD in file and run node Racro.js

for creating discord bot change values in index.js (leave the .env file blank)
for dhooking put your webhook to line 184
